자바에서 javac.exe 와 java.exe 파일이 nin 디렉터리 안에 있는데
이를 어디서든 사용하기 위해서 Path 설정을 해야한다.
환경 변수에서 상단에 변수명 중 PATH 가 존재하는데 편집을 누르고 변수 값을 추가해준다.
(bin 까지의 경로를 추가 입력해준다.)


자바의 프로그램 실행 구조


일반 적으로 운영체제 위에서 프로그램이 돌아가지만
자바는 Java Virtual Machine(OS 위의) 위에서 프로그램이 돌아간다

-> 운영체제와 상관없이 프로그램이 돌아가게 만듬. C등은 OS 에 따라서 약간씩 다르게 프로그래밍
  을 해야하지만 JAVA 는 하나의 코드를 여러 운영체제에서 사용이 가능하다.
   (내가 알기로는 운영체제에 따라서 심지어 메모리를 읽어오는 방향?순서? 도 약간씩 다른 것으로 암)
-> 운영체제에 따라서 키보드 그래픽 마우스 관련 기능이 다르게 제공됨. 그래서 서로 다르게 코딩을 해야함
  하지만 자바 버츄얼 머신이 이를 커버줌


자바 컴파일러는 자바가상머신이 이해할 수 있는 코드를 생성해냄.java -> .class(바이트 코드)
(컴파일러 :javac .exe)
(자바 런처:java .exe)
